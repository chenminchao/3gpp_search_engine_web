5.2.5      Charging Function domain CDRs

5.2.5.1          General

This subclause contains the syntax definitions of the CDRs for the CHF.

5.2.5.2          CHF CDRs

This subclause contains the abstract syntax definitions that are specific to the CHF CDR types defined in this document.

.$CHFChargingDataTypes {itu-t (0) identified-organization (4) etsi (0) mobileDomain (0) charging (5) chfChargingDataTypes (15) asn1Module (0) version1 (0)}
