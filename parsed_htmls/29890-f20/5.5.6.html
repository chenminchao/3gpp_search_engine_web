A.16 Sessions multiplexing over a single transport connection

HTTP/2 supports multiplexing multiple parallel requests in separate streams in a non-blocking fashion at HTTP level) over the same TCP connection.

QUIC/HTTP/2 supports multiplexing multiple parallel requests in separate streams in a non-blocking fashion over the same UDP connection.
